# CUBE GAMEEEEEEEEEE

A runner-esque game where you control a cube and attempt to avoid running into obstacles.

there's only one level so not much to worry about in terms of difficulty.

this project is unlikely to be updated any further.

## How to install

1 - go to the releases tab in github and download the zip.
2 - unzip the package and navigate to the folder containing your OS
3 - run the executable called "cube-game"
4 - enjoy

## How to play

move left and right with the A and D keys and try to avoid the blue rectangles/cubes
